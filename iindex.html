The Big Differences

Encoding

Goal: Format Data

Reversible? Yes

Key? No

Ex: Base64

Hashing

Goal: Integrity / ID

Reversible? No

Key? No (usually)

Ex: SHA-256

Encryption

Goal: Secrecy

Reversible? Yes (with key)

Key? Yes

Ex: AES, RSA

J.W.T.

JSON Web Token



It's a way to securely transmit information between parties as a JSON object.

It acts like a Digital Passport. The server stamps it, hands it to you, and you show it every time you want to enter a restricted area (API).

It uses Encoding AND Hashing/Encryption!

The Structure

A JWT is just a long string separated by dots:
